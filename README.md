# Lab M1.08 - Linux for Cloud Engineering

## Learning Objectives
- SSH into cloud instances securely
- Analyze system logs for troubleshooting
- Manage system resources (disk, memory, processes)
- Install and configure software packages
- Set up web server with proper permissions
- Use environment variables and configuration files

## Key Findings
- Network configuration (Internet Gateway, route tables, security groups) is critical for SSH access
- Nginx is easy to install and configure on Ubuntu
- System logs provide valuable information for troubleshooting
- Environment variables and aliases improve productivity
